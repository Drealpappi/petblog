# petblog
Django 6.0 Pet Blog | Python 3.14 &amp; PostgreSQL A full-stack blogging platform featuring custom user authentication, a secure contact system with real-time email notifications, and the Django Messages framework. Migrated from SQLite to a production-grade PostgreSQL database for enhanced data integrity and scalability.
